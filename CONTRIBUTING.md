# Contributing to GENIA Americas

## Workflow

1. Fork the repository and clone your fork locally.
2. Create a branch using this convention:
   - dataset/<username>-<region>
   - research/<short-description>
   - docs/<short-description>
   - fix/<short-description>
3. Make your changes following the guidelines for your contribution type.
4. Commit with a clear, descriptive message in English.
5. Push your branch and open a pull request against main.
6. Reference the relevant issue number in your PR description.

For AmeriNLI submissions, see datasets/amerinli/README.md for format requirements.

## Review Process

Pull requests are reviewed weekly. A maintainer will respond within 7 days.
All merged dataset contributions are acknowledged in the release notes.

## Attribution

By submitting, you agree to license your contribution under Apache 2.0 and
confirm you have the right to do so. Contributors retain attribution.

## Questions

Open an issue or contact admin@genia.ai.
