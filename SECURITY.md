# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this repository or any GENIA Americas platform, please report it responsibly.

**Do not open a public GitHub issue.**

Contact us directly:
- Email: admin@genia.ai
- Subject line: `[SECURITY] Brief description`

We will acknowledge your report within 48 hours and aim to resolve confirmed issues within 14 days.

## Scope

This repository is primarily documentation and research. Security reports relevant to:
- The Glapagos platform (glapagos.com)
- API specifications in `platform/api/`
- Any tooling in `tools/`

## Out of Scope

- Theoretical vulnerabilities without proof of concept
- Issues in third-party dependencies not maintained by GENIA Americas
